# MovieMap

This is a demo app using:
 * Leaflet
 * AngularJS
 * Express
 * Bower
 * Freebase
 * OpenMap

## Install

```shell
mkdir myproject
cd myproject
git clone https://github.com:jewelsjacobs/moviemap.git .
grunt install
```
